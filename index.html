Local Arcade Preview

Test your games in a boxed web portal before you publish them.

This works both from your computer and from GitHub Pages. Click a game card and you'll get a small embedded player with fullscreen plus a full-view fallback on the next screen.

no hosting needed local game folders github pages ready

Setup

  1. Put each HTML game in its own folder inside games/.
  2. Add one entry to games/catalog.js.
  3. Open index.html locally or publish this folder to GitHub Pages.
